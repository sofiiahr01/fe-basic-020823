Селектори

  • A - Селектор елементу - може виступати будь-який елемент HTML, для якого визначаються правила форматування, такі як: колір, фон, розмір тощо.
  • #id - Ідентифікатор - визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього через скрипти.
  • .class - Класи - застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки. Елементи при цьому можуть бути різними, але мати частково або повністю схожі елементи оформлення.
  • .class1.class2 - Мультикласи - до будь-якого елемента одночасно можна додати кілька класів, перераховуючи їх в атрибуті class через пробіл. У цьому разі до елемента застосовується стиль, описаний у правилах для кожного класу.
  • * - Універсальний селектор - використовується, коли потрібно встановити одночасно один стиль для всіх елементів веб-сторінки, наприклад, задати шрифт або накреслення тексту.
  • A B - Вкладені селектори - використовується для коректного використання стилів під час вкладання одних елементів всередину інших.
  • A > B - Дочірні селектори - елемент, який безпосередньо розташовується всередині батьківського елемента. Стиль застосовується до елемента В, тільки коли він є дочірнім елементом А.
  • A + B - Сусідні селектори - елементи веб-сторінки, коли вони йдуть безпосередньо один за одним у коді документа.
  • A ~ B - Споріднені селектори - елементи веб-сторінки, коли вони йдуть безпосередньо один за одним у коді документа. A ~ B - Споріднені селектори - за своєю поведінкою схожі на сусідні селектори, але на відміну від них стильові правила застосовуються до всіх прилеглих елементів. Наприклад, для селектора h1~p стиль буде застосовуватися до всіх елементів p, що розташовуються після заголовка h1. При цьому h1 і p повинні мати спільний батьківський елемент, тому якщо p вставити всередину div, то стилі застосовуватися вже не будуть.
  • [attr] - Селектори атрибутів - встановлює стиль для елемента, якщо задано специфічний атрибут. Його значення в цьому випадку не важливе.
  • [attr='value'] - [атрибут="значення"] - встановлює стиль для елемента в тому випадку, якщо задано певне значення специфічного атрибута.
  • [attr^='value'] - [атрибут^="значення"] - встановлює стиль для елемента в тому випадку, якщо значення атрибута починається із зазначеного тексту.
  • [attr$='value'] - [атрибут$="значення"] - встановлює стиль для елемента в тому випадку, якщо значення атрибута закінчується зазначеним текстом.
  • [attr*='value'] - [атрибут*="значення"] - визначає, що значення атрибута містить зазначений текст.
  • [attr~='value'] - [атрибут~="значення"] - задати стиль за наявності в списку необхідного значення, коли значення атрибутів перераховуються через пробіл.
  • [attr|='value'] - [атрибут|="значення" - стиль застосовується до елементів, у яких атрибут точно дорівнює зазначеному значенню або починається із зазначеного значення, після якого йде дефіс.
