• git add - індексує вказаний файл для наступного коміта.
  • git commit -m "[повідомлення з описом]" - фіксує проіндексовані зміни та зберігає їх у історію версій.
  • git status - передивитись зміни у репозиторії.
  • git push - завантажує усі зміни локальної гілки у віддалений репозиторій.
  • git pull - завантажує історію з віддаленого репозиторія та об'єднує її з локальною.
  • git checkout - переключення між різними версіями цільового об'єкта.
  • git branch - список іменованих гілок з зазначенням обраної гілки.
  • git reset - відміняє індексацію вказаного файла, при цьому зберігає його вмістиме.
  • git rebase - об'єднати зміни, зроблені в одній гілці, з іншою гілкою.
  • git log - історія комітов для поточної гілки.
